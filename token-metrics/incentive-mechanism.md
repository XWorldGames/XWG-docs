# 激勵制度

X World Games 的激勵制度大致分為以下幾類：\


#### 遊戲激勵

有一定比例的憑證將保留用於玩家遊戲激勵。在遊戲中，玩家可藉由參與各種不同的遊戲行為獲得不同數量的XWG獎勵。此設計可以鼓勵用戶參與遊戲，擴大社區並保持吸引力。

#### 開發激勵

X World Games 將為廣大的遊戲社區建設者及信仰者提供一個展示自我的平台。遊戲開發者可在平台上製作獨一無二的遊戲角色，裝備及道具，未來還將提供一系列的開源遊戲工具供開發者製作獨立遊戲項目，豐富遊戲多元化生態。開發者在成功創造後將會依程度獲得XWG作為激勵。\


#### 流動性激勵

X World Games 將在遊戲生態中提供各種DeFi 去中心化金融功能，包括流動性挖礦，質押挖礦， DeFi借貸抵押及Swap。玩家在成為做市商並且為XWG提供流動性成功後也可得到XWG獎勵。\


#### 治理激勵

擁有XWG的玩家可通過提案和投票等去中心化自治組織(DAO)行為來參與社區及遊戲治理，來決定存平台操作和遊戲開發相關的問題，並根據XWG持有的數量來確定。為了鼓勵用戶參與更多的投票過程，成功投票後也將得到XWG獎勵。

#### 社群激勵

X World Games 將會在遊戲上線、或交易所等重大事件前後，舉辦一些營銷宣傳活動來擴大玩家參與和社區影響，如空投，預售，遊戲競賽，賞金活動等計劃。詳細公告將會通過官網以及其他官方社交平台如Twitter、Discord、Medium等進行公佈。\
