# Vision

![](../../.gitbook/assets/x-eco.jpeg)

The vision of X World Games is to build a diversified gaming ecosystem that connects traditional game players and the blockchain world. 

We will build the gaming infrastructure to help more game developers who want to enter the crypto space to create the underlying blockchain framework.

X world will be [**open world** ](https://en.wikipedia.org/wiki/Open_world)to all players,creators and developers to play and earn.
