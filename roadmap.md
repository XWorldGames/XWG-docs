# 路線圖

X World Games 将会在未来数年内完成一项的庞大平台基础架构计划，包括游戏聚合中心，NFT交易所，DeFi交易机制，跨游戏协议端口和全功能游戏创建工具箱。 同时，我们将通过与外部合作伙伴或内部开发来发布更多款游戏，并签署更多正版动漫IP来丰富我们的游戏生态系统和社区，将帮助这些移植游戏与我们的凭证，矿池和经济系统连接起来，并指引我们的社区迈向一个全面的去中心化治理。

X World Games 將會在未來數年內完成一項的龐大平台基礎架構計劃，包括遊戲聚合中心，NFT交易所，DeFi交易機制，跨遊戲協議端口和全功能遊戲創建工具箱。同時，我們將通過與外部合作夥伴或內部開發來發布更多款遊戲，並簽署更多正版動漫IP來豐富我們的遊戲生態系統和社區，將幫助這些移植遊戲與我們的憑證，礦池和經濟系統連接起來，並指引我們的社區邁向一個全面的去中心化治理。
