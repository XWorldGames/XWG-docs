X World Games是下一代去中心化區塊鏈遊戲生態系統。 “X World”代表未知有待探索的的元宇宙世界觀，“Games”則代表更多的遊戲會加入到這個生態裡來。 X World Games 的使命是將區塊鏈技術帶入一個開放的遊戲世界，讓世界各地的玩家可以一起參與遊戲； 簡單地說，X World Games 將把區塊鏈帶入一個充滿樂趣的世界。

X World Games不僅是區塊鏈玩家的遊戲， 同時也是一款可以將區塊鍊和成千上萬玩家結合的遊戲。 X World Games現階段正與眾多區塊鏈遊戲廠商洽談合作，吸引更多傳統遊戲玩家，拓展其生態，旨在做區塊鏈遊戲領域的“Open Sea”，成為區塊鏈遊戲的聚合器。
