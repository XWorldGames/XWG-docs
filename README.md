X World Games是下一代去中心化区块链游戏生态系统。“X World”代表未知有待探索的的元宇宙世界观，“Games”则代表更多的游戏会加入到这个生态里来。X World Games 的使命是将区块链技术带入一个开放的游戏世界，让世界各地的玩家可以一起参与游戏； 简单地说，X World Games 将把区块链带入一个充满乐趣的世界。

X World Games不仅是区块链玩家的游戏， 同时也是一款可以将区块链和成千上万玩家结合的游戏。X World Games现阶段正与众多区块链游戏厂商洽谈合作，吸引更多传统游戏玩家，拓展其生态，旨在做区块链游戏领域的“Open Sea”，成为区块链游戏的聚合器。
