# 問與答

### 什麼是 $XWG 代幣？

$XWG 通證是 X World Games 的原生代幣，基於幣安智能鏈的 BEP-20 標準構建，固定總供應量為 100 億。&#x20;

$XWG 是推動 X World Games 生態系統發展的重要燃料。 $XWG 持有者將能夠通過參與遊戲挖礦機制、DeFi 運營和 DAO 投票過程來領取獎勵。&#x20;

$XWG 也將被接受作為 X World Games Marketplace 的支付貨幣，玩家可以在這裡買賣他們的遊戲內 NFT 資產。

### X World Games 的系統要求是什麼？

任何帶有 Chrome/Firefox 瀏覽器的 PC/Mac 或任何移動 iOS/Android 設備。



### 我可以使用移動設備在 X World Games 上玩遊戲嗎？

是的，我們有支持移動設備的遊戲，但有些不支持。

### 我可以從多個設備登錄遊戲嗎？

是的，沒有設備數量限制，只要每個登錄都有一個錢包地址。



### 我可以使用不同的瀏覽器嗎？

雖然在技術上可以使用其他瀏覽器，但我們仍然建議您使用 Chrome 或 Firefox 以獲得最佳遊戲體驗。



### 我需要一個加密錢包來玩遊戲嗎？

是的，需要像 [MetaMask(小狐狸)](https://metamask.io)或[幣安錢包](https://www.binance.com/en/wallet-direct)這樣的加密錢包。錢包地址作為您的賬戶，確保您所有數字資產（如XWG代幣、NFT卡、虛擬道具）的安全和在虛擬世界中的進步。&#x20;

如果您目前沒有加密錢包，您可以先使用手機號或郵箱註冊並登錄遊戲，您的遊戲內數據將保存在您的本地計算機上（鏈下）。但是，當您需要使用您的遊戲資產進行鏈上操作時，您仍然需要將您的帳戶與加密錢包地址綁定。



### 我把我的錢包弄丟了！我的帳戶會怎樣？

如果您無法訪問您的錢包，您將無法訪問存儲在您帳戶中的所有通證和 NFT 資產，我們也無能為力。因此，請始終將您的錢包恢復詞保存在安全可靠的地方。
